소개

안녕하세요, 저는 5년 경력의 웹 개발자 김민수입니다. 프론트엔드와 백엔드 개발에 모두 능숙하며, 사용자 중심의 웹 애플리케이션을 개발하는 것을 좋아합니다.

서울대학교에서 컴퓨터 공학을 전공하였으며, 다양한 프로젝트 경험을 통해 실무 역량을 키워왔습니다. 항상 새로운 기술을 배우고 적용하는 것에 열정을 가지고 있습니다.

프로젝트를 진행할 때는 코드 품질과 가독성을 중요시하며, 팀원들과의 원활한 협업을 위해 노력합니다. 저의 목표는 사용자에게 최적화된 웹 경험을 제공하는 것입니다.

기술

HTML5

HTML5

시맨틱 마크업과 접근성을 고려한 구조적인 HTML 코드를 작성합니다.

CSS3

CSS3

반응형 웹 디자인과 애니메이션을 포함한 현대적인 스타일링을 구현합니다.

JavaScript

JavaScript

ES6+ 문법을 활용한 인터랙티브 웹 기능과 SPA 개발에 능숙합니다.

React

React

컴포넌트 기반 아키텍처와 상태 관리를 활용한 모던 UI 개발이 가능합니다.

Node.js

Node.js

Express 프레임워크를 활용한 RESTful API 개발과 서버 구축에 능숙합니다.

Database

데이터베이스

MongoDB, MySQL 등 다양한 데이터베이스 시스템을 활용할 수 있습니다.

프로젝트

온라인 쇼핑몰

React와 Node.js를 활용한 풀스택 이커머스 웹사이트입니다. 사용자 인증, 상품 검색, 장바구니 기능을 구현했습니다.
